# Get available years for Utah enrollment data

Returns the range of years for which enrollment data is available from
the Utah State Board of Education.

## Usage

``` r
get_available_years()
```

## Value

Integer vector of available years

## Examples

``` r
get_available_years()
#>  [1] 2014 2015 2016 2017 2018 2019 2020 2021 2022 2023 2024 2025 2026
```
